# The Cat APIを試す

## 実行環境
ローカルの  `debian 11.7`

## 実行
1.  cats.js の `YOUR_ACCESS_TOKEN` を置き換え
```
const accessToken = 'YOUR_ACCESS_TOKEN';  // YOUR_ACCESS_TOKEN は置き換え
```
  
2. .github/workflows 配下で GitHub Actions を定義
3. コマンド実行
    * npm はインストール済
```
# npm 有効化
npm init -y
# パッケージインストール
npm i axios
```
